Website Development Process


The following are snapshots of the prototyping and design process that was involved in the creation of my personal website. While most of the work is not shown, the development of this website followed the iterative design process and a rapid prototyping model that migrated from low fidelity prototype to medium and finally high fidelity Website prototypes. The website is currently on its 4th iteration and first published version (version 1.0).



Creation of the Webite... Beginning with a process model



To provide a platform to:

  • Advertise the skill uniquness and personal diversity of Sammy Robens-Paradise
  • Describe the above to possible employers or those who may be of interest




The Website must:

  • be captivating to users and feature UI that simply displays the technical skillset of Sammy Robens-Paradise
  • Posess Multi-platform functionality to be displayed on both mobile phones and desktop computers
  • Leave visitors with a good impression and not deter possible clients




Scope:

  • The website will be written in: html, CSS, JavaScript/JQuery framework
  • The website willuse the following frameworks: bootstrap 4, Font awesome, bootstrap social, JQuery 3.3.1
  • The website will use Github's domain and hosting service at SammyRobensParadise.github.io
  • The website will use github's source control and branching tools for the development process
